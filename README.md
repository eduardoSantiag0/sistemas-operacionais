## Sistemas Operacionais

### Objetivos da disciplina: 

- Compreender os conceitos, objetivos e funcionamento dos sistemas operacionais
- Entender o gerenciamento de processos, threads, recursos e armazenamento de dados
- Conhecer algoritmos e abstrações usados em sistemas operacionais
- Identificar requisitos de diferentes tipos de sistemas (tempo real, servidores, dispositivos limitados)
- Utilizar de forma eficiente os recursos e serviços do sistema operacional
- Desenvolver habilidades para projetar e implementar sistemas operacionais
